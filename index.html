Computer Vision • 3D Modeling

By analyzing the parallax differences between the two object views shown below, we can computationally reconstruct spatial depth and surface geometry. This technique is fundamental to my work in remote sensing and was a core component of my thesis on detecting runway hazards.

Dense Point Cloud Generation

To better visualize the surface topology and elevation changes, the raw 3D point cloud data was processed further. By interpolating the Z-coordinates (depth) of the generated points, we can project the geometry onto a 2D plane using isolines. The image below displays the converted 3D points as a contour map, effectively highlighting the gradients and surface morphology of the scanned object.
