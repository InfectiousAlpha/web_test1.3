Stereophotogrammetry Reconstruction

Computer Vision • Python • 3D Modeling

This project demonstrates the application of stereophotogrammetry using Python algorithms to create high-fidelity 3D models from standard 2D imagery. By analyzing the parallax differences between the two object views shown below, we can computationally reconstruct spatial depth and surface geometry. This technique is fundamental to my work in remote sensing and was a core component of my thesis on detecting runway hazards.

Surface Contour Testing using Water

Timelapse Analysis • Binary Processing • Water Modeling

In this experimental phase, the reconstructed surface model is subjected to physical testing. The process involves wetting the surface and capturing a timelapse of the drying process until completely dry. These images are processed into binary maps to isolate water coverage, allowing for the generation of contour maps that represent water height and retention patterns over time.
